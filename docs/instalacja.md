# Instalacja

## Wymagania

- Kompilator GCC lub Clang
- System Windows, Linux lub macOS

## Pobranie projektu

```bash
git clone https://github.com/MyNameIsTomek/peojektPracaGrupowa.git
cd peojektPracaGrupowa
```

## Kompilacja

```bash
gcc main.c -o projekt
```

## Uruchomienie

Windows:

```powershell
projekt.exe
```

Linux i macOS:

```bash
./projekt
```
